# Simple Linear Regression for Sales Prediction based on 'TV' Marketing Budget

___

This repository contains a simple implementation of linear regression to predict sales based on the marketing budget allocated to TV advertisements.

## Dataset
The dataset used for this project is provided in the data directory. It contains two columns: ***TV*** (the amount spent on TV advertising) and ***Sales*** (the corresponding sales generated). This dataset is used to train the linear regression model.

## Dependencies
To run this project, you need to have the following dependencies installed:

- Python 3.x
- NumPy
- pandas
- matplotlib
- scikit-learn

## Results
After running the script, you will see the coefficients of the linear regression model printed, indicating the relationship between TV advertising budget and sales. Additionally, the script will generate a plot showing the original data points along with the regression line.
